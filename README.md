# bitlocker-tpm-and-usb-demo
This repository contains a practical demonstration of **BitLocker encryption** on Windows, including methods with and without TPM 2.0. You can follow the steps to secure your drives effectively.
## Features
- Encrypt the entire drive or used disk space
- Set up password protection
- Enable BitLocker without TPM 2.0
- Step-by-step instructions with screenshots
